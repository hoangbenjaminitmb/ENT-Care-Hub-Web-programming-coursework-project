ENT Care Hub

What ENT Care Hub is?

ENT Care Hub is a website created for my web programming coursework at Loughborough University. Started in Early March and finished in Early May, it is designed to help individuals identify suitable health consultants based on specific criteria, such as availability of the consultant, finding nearest clinics, and choosing a consultant based on reviews.

Throughout my project, I have used different JavaScript and CSS libraries to help complete this project, along with assistance for areas I have struggled with. Below is a list of the libraries I have used for my coursework.

Libraries I have used

The following JS libraries I have used to complete my coursework includes:

Leaflet.js (nominatim)

I used leaflet.js to help convert the clinics' latitude and longitude on phpMyAdmin to nearby addresses. In addition, I also used it to get a map showing the location of clinics

Bootstrap.js

Bootstrap.js was used to help with the styling of the navigation links on enthub.php, the header and footer, as well as the button styling

In addition to the JS libraries I have used, I have also used the following CSS libraries

Bootstrap.css

Bootstrap.css was used with the styling of text, the navigation links, etc. It is supplemented by Bootstrap.js



APIs used

The APIs that were used in my web programming coursework included:

Google Fonts API

The Google Fonts API was used to help stylise the subheadings, page names, and the logo. The font I have used for my coursework is Lexend



Other tools used

Other tools I have used for my coursework include:

Microsoft Copilot

I used Microsoft Copilot to help with MySQLi query fixes, plus JavaScript fixes, including the debugging of JS code. I also used it to help with design ideas.
